🎮 CODE QUEST — Apprends à coder en jouant
créé par Shizou

CODE QUEST est une application web interactive conçue pour initier les débutants aux concepts fondamentaux de la programmation.
À travers une interface inspirée des jeux vidéo, l'utilisateur progresse de la logique en français vers le véritable code JavaScript.

✨ Fonctionnalités
Apprentissage Progressif : Un parcours qui commence en français pour comprendre la logique, puis transitionne doucement vers l'anglais (JavaScript).

Éditeur de Code Intégré : Un mini-interpréteur qui permet d'écrire et de tester son code directement dans le navigateur.

Système de Gamification :

Points d'Expérience (XP) à chaque mission réussie.

Niveaux de joueur pour suivre sa progression.

Séries (Streaks) pour encourager la réussite consécutive.

Compagnon Interactif : Nono le robot, ton guide personnel qui explique chaque concept avec humour et clarté.

Responsive Design : Une interface moderne et sombre (Dark Mode), optimisée pour le confort visuel des développeurs.

📚 Curriculum du Parcours "Les Bases"
Le logiciel couvre les piliers de l'algorithmique :

Variables : Le concept de stockage (la boîte).

Calculs : Manipuler les nombres et les opérateurs.

Conditions : La logique si / sinon (if / else).

Boucles : Répéter des tâches efficacement (for).

Fonctions : Créer ses propres commandes réutilisables.

Le Boss Final : Le célèbre défi FizzBuzz pour valider ses acquis.

🛠️ Technologies utilisées
Ce projet est une application "Single File" ultra-légère utilisant :

HTML5 : Structure sémantique.

CSS3 : Animations personnalisées (fadeUp, robotBob), variables CSS et Flexbox/Grid.

JavaScript (ES6+) : Logique de l'application, moteur d'interprétation du code et gestion de l'état du jeu.

Google Fonts : Polices Fredoka pour le côté jeu et Fira Code pour la partie technique.

🚀 Installation et Utilisation
Aucune installation complexe n'est requise.

Télécharge le fichier code-quest.html.

Ouvre-le dans n'importe quel navigateur moderne (Chrome, Firefox, Edge, Safari).

Commence ta quête !
